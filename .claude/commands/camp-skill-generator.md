# Camp Skill Generator

강의 교안 마크다운 파일을 읽고, 해당 강의를 인터랙티브하게 진행할 수 있는 Claude Code 스킬(slash command)을 생성합니다.

## 입력

사용자가 지정한 강의 교안 파일 경로. 지정하지 않으면 프로젝트 내 `day*/day*-lecture.md` 패턴의 파일 목록을 보여주고 선택하게 합니다.

## 실행 절차

### 1단계: 교안 파일 분석

지정된 강의 교안 마크다운 파일을 읽고 다음을 파악합니다:

- **강의 제목**: 문서의 `# 제목`
- **대상 청중**: 대상 섹션의 내용
- **섹션 구조**: 각 섹션의 번호, 제목, 소요 시간, 핵심 내용
- **실습 항목**: 실습 단계와 구체적인 명령어
- **퀴즈**: 퀴즈 문제와 정답
- **준비물/사전 조건**: 사전에 필요한 설정이나 지식

### 2단계: 스킬 생성

분석한 내용을 기반으로 `.claude/commands/` 디렉토리에 스킬 파일을 생성합니다.

생성할 스킬 파일명 규칙: `camp-{day번호}.md` (예: `camp-day1.md`)

#### 생성되는 스킬의 구조

```markdown
---
description: {강의 제목} - 인터랙티브 강의 진행
---

# {강의 제목}

이 스킬은 "{강의 제목}" 강의를 인터랙티브하게 진행합니다.

## 대상
{교안의 대상 정보}

## 진행 방식

강의를 섹션별로 진행합니다. 각 섹션에서:
1. 핵심 개념을 설명합니다
2. 실습이 있으면 사용자와 함께 직접 실행합니다
3. 퀴즈가 있으면 사용자에게 문제를 내고 답을 확인합니다
4. 사용자가 준비되면 다음 섹션으로 넘어갑니다

## 섹션별 진행 가이드

{각 섹션의 상세 진행 가이드}

## 진행 규칙

- AskUserQuestion 도구를 사용하여 퀴즈를 출제하세요
- 실습 단계에서는 사용자가 직접 명령어를 실행하도록 안내하되, 막히면 도와주세요
- 각 섹션이 끝나면 AskUserQuestion으로 "다음 섹션으로 넘어갈까요?"를 확인하세요
- 사용자가 질문하면 교안 내용을 기반으로 답변하세요
- 교안에 없는 내용이면 솔직히 말하고, 관련 레퍼런스를 안내하세요
```

### 3단계: 결과 보고

생성된 스킬 파일의 경로와 사용법을 안내합니다:

- 생성된 파일 경로
- 실행 방법: `/camp-{day번호}` 로 실행
- 생성된 스킬의 섹션 요약

## 주의사항

- 교안의 내용을 그대로 복사하지 말고, **인터랙티브 진행에 적합한 형태**로 변환하세요
- 퀴즈는 정답을 스킬 안에 포함하되, 사용자에게는 AskUserQuestion으로 먼저 답하게 하세요
- 실습 명령어는 그대로 보존하되, 단계별로 나누어 진행하도록 구성하세요
- 각 섹션의 예상 소요 시간을 유지하세요
- 생성 후 사용자에게 스킬 내용을 검토할지 물어보세요
