# Day 4: 스킬 배포와 확장 — 플러그인에서 Agent SDK까지

이 스킬은 Day 4 강의 핸드아웃(scrollytelling HTML)을 브라우저에서 엽니다.

## 실행

1. Bash 도구로 `open` 명령어를 실행하여 `day4/day4-handout.html` 파일을 브라우저에서 엽니다.

```bash
open day4/day4-handout.html
```

2. 열린 후 사용자에게 안내합니다:

> Day 4 강의 핸드아웃이 브라우저에서 열렸습니다.
>
> - **스크롤** 또는 **키보드 ↑↓**로 슬라이드 이동
> - 우하단 **버튼 탭**으로 섹션 목차 열기
> - URL에 `#slide-N`을 붙이면 해당 슬라이드로 바로 이동 (예: `#slide-38`)
>
> 총 53개 슬라이드, 6개 섹션 (90분)
